
# Инфрострукторное представление

Функционал  Облочное решение

1. База данных - **PostgreSQL** 
2. Аудентификация, при необходимости если появится то bff для мобильных устройств и отдельно для веб-версий, распределение нагрузки **API Gateway**
3. Все должно быть развернуто с помощью kubernetes (он же должен реализовать быстрый подъем на другом ЦОДе где уже хранится реплика данных наших приложений)

# Безопасность

1. API Gateway обеспечивают аутентификацию.\
			реализовывают при необходимости bff подход (фасад)
            обеспечивают load-balancing
            следят за трафиком
            логируют вызовы АПИ сервисов
2. Каждый сервис обязательно реализует паттерн ZeroTrust
3. Каждый МС перепроверяет есть ли у УЗ доступ не только до методов, но так же до информации конкретного клиента по которому идет запрос.\
4. Должна быть настроена система мониторинга подозрительных событий(ввод несколько раз неверных паролей, быстрая смена геолокации пользователя), сквозная тресировка пользовательских запросов.
